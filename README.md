# Keyboard - Nexss PROGRAMMER 2.0

Handling keyboard

## Examples

```sh
nexss Keyboard --type="ls -la"  # writes ls -la
nexss Keyboard --type="#s" # WIN + s
nexss Keyboard --type="#d" # WIN + d / Show Desktop
nexss Keyboard --type="#b" # WIN + b / Show Taskbar

```

## Special Keys

### Windows - Auto Hotkey

```ahk
! = ALT
+ = SHIFT
^ = CTRL
# = WIN
```
